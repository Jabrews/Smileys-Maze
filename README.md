# Smileys Maze

A fast-paced horror maze where you explore dark, shifting floors while collecting papers and avoiding hostile workers. Inspired by Baldi’s Basics and SCP: Containment Breach, the game focuses on tension, navigation, and quick decision-making. Traverse three floors of the building, using vents and stairwells to escape danger and outmaneuver threats.
